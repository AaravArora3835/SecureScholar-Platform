🔒 Module 1 — Password Security

🎯 Goal: Learn to create and manage strong passwords safely.

Overview

This module teaches the foundations of password security — why strong passwords matter, how to make them stronger, and how to protect accounts using password managers and two-factor authentication (2FA).

Learning Objectives

  • Understand what makes a password strong.
  • Identify and avoid weak or common passwords.
  • Learn how password managers keep your credentials safe.
  • Enable 2FA to protect high-value accounts.
